PROJECT TITLE: Tarea Programada 3

PURPOSE OF PROJECT: Dominio de creación de estructuras dinámicas como listas y
árboles binarios y creación de una interfaz gráfica interactiva para procesamiento y
visualización de los datos procesados.
VERSION or DATE: 9/12/2020
HOW TO START THIS PROJECT: Para dar comienzo al programa tendrá que copilar el Main de la siguiente manera: javac Main.java, luego procedemos al "java Main [nombre del archivo del texto(txt)]", si por lo contrario no se ha insertado un archivo de texto, se hará por default, en donde se asignará automáticamente el archivo de texto haru.txt

COMIENZO DEL PROGRAMA:
El programa se propone de estructuras dinámicas necesarias para
contar y ordenar por frecuencia las palabras almacenadas en un archivo de texto y representarlas en estructuras diferentes
Al iniciar el programa aparecerá una ventana en la cual contiene 4 botones:

-Botón Lista Frecuencia

Aparece una lista que clasifica las palabras de mayor a menor frecuencia y después alfabéticamente

-Botón Árbol Binario

Aparece una representación, al analizar el archivo del texto, en forma de árbol binario donde cada rama será lo más proporcional posible en lo referente a la suma de las frecuencias de las palabras en cada rama.

-Botón Mapa

Aparece una representación, al analizar el archivo del texto, en forma de mapa de la superficie asociada a la información que contiene el árbol binario. En cada
cuadrante el área deberá ser equivalente a los datos en las diferentes ramas del dicho árbol

-Botón Salir

Sale del programa
